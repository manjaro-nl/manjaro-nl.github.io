Registreer je eigen nickname

Indien gewenst, kan je je eigen naam registreren op irc.

Verander je naam eerst in je nieuwe naam.

/nick nieuwenaam

Registreer je nieuwe naam.

/msg NickServ REGISTER paswoord emailadres

Een email met een bevestigingscode wordt gestuurd naar je emailadres. Bevestig je account met de toegestuurde code.

/msg nickServ CONFIRM code

Je nickname veiligstellen

Indien je wil dat niemand anders jouw nickname gebruikt, doe dan:

/msg NickServ SET ENFORCE ON

Nu kan niemand je nickname misbruiken. Kijk na of je account goed geconfigureerd is:

/msg NickServ IDENTIFY nickname paswoord

Paswoord vergeten?

Indien je je paswoord vergeten bent, wordt een nieuwe sleutel naar je email gestuurd met:

/msg NickServ SENDPASS nickname

Uiteindelijk verander je paswoord.

/msg NickServ SETPASS nickname sleutel paswoord

Tips en Tricks

Verberg je emailadres voor andere gebruikers:

/msg NickServ SET HIDEMAIL ON

Indien door internetproblemen de server denkt dat je nog steeds verbonden bent, kan je de sessie met je nickname geforceerd afsluiten: 

/msg NickServ GHOST nickname paswoord
